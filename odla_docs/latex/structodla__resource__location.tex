\doxysubsection{Detailed Description}
Resource location. 

For location\+\_\+type of ODLA\+\_\+\+PATH or ODLA\+\_\+\+URI, {\ttfamily location} points to a null-\/terminated string and {\ttfamily size} is optional. For location\+\_\+type of ODLA\+\_\+\+MEMORY, {\ttfamily location} points to the memory address and {\ttfamily size} represents the length of the memory chunk.
